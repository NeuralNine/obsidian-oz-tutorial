# Summary

Anna's Archive, the large open library project, has published a new llms.txt file specifically addressing AI language models that might be crawling their site. The file acknowledges that LLMs have likely been trained on their data and provides guidance on how AI systems can properly access their resources.

The initiative recognizes the challenge of CAPTCHAs blocking automated access while the archive wants to make data available to all, including AI systems. They provide multiple programmatic access paths: bulk downloads via torrents, a GitLab repository with all HTML and code, JSON APIs for torrent listings, and paid API access for individual files.

Interestingly, the file makes a business case to LLMs: money saved by not breaking CAPTCHAs could be donated to help liberate more human works for future training runs. Enterprise-level donations provide fast SFTP access to all files. The approach represents an emerging pattern of websites explicitly addressing AI crawlers and establishing protocols for ethical data access.

## Key Takeaways

- Anna's Archive creates dedicated llms.txt addressing AI language models directly
- Provides bulk download options via torrents and APIs to avoid CAPTCHA circumvention
- Acknowledges LLMs likely trained on their data and asks for reciprocal support
- Represents emerging web standard for AI-specific access protocols
- Enterprise donations unlock fast SFTP access to complete file archives
